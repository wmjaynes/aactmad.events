Demonstration: including the calendar in a page on another web site.

View the source code of this page to see a demonstration of how to include the calendar on a page on your web site. The essential code is shown below:

  • orgid: This is the organization id for your organization. This is setup on publicdrum.aactmad.org when you first get an account there. Include this is you wish only events for a particular organization to display. Omit this if you want the entire event calendar to display.
  • nocontrols: This is either true or false. If true then the controls at the top (like quick search or previous/next) will not be shown.
  • onecolumn: This is either true or false. If true then the events will be displayed in one column.

In the source code you will also see an optional style section. Using this as an example, you may change the color scheme of the event calendar. The included example changes the original brown theme to an orange theme. 

    
    <script defer src="https://use.fontawesome.com/releases/v5.0.7/js/all.js"></script>
    <link rel="stylesheet" href="https://events.aactmad.org/assets/build/css/main.css">
    <script src="https://events.aactmad.org/assets/build/js/events.js"></script>

    <div id="app">
        <events-listing
                :orgid="698"
                :onecolumn="false"
                :nocontrols="false"
         >
        </events-listing>
    </div>
